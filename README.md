# #ToDo App 

## General info:
A classic todo application that allows to: 
register, 
log in, 
add new tasks, 
make them done/undone,
delete them.
Application development is still in progress. 

## Technologies: 
ASP.NET Core Blazor, 
Bootstrap 4.3.1, 
Radzen

## Setup:
To run the project, install it locally.
